# SKILL 04 — Writing Modes & Frameworks

## Purpose
This skill generates the post body after the hook (Skill 03) is selected. It contains all 8 writing modes, each with multiple rotating frameworks. The system reads the insight and selects the framework that best fits — not always the same one. This prevents every post in the same mode from sounding identical.

The goal of every post: the reader finishes and thinks "this person lives inside this industry" — not "this person read the news today."

---

## Trigger
Activates after:
1. User selects a theme (Skill 02)
2. Hook is chosen (Skill 03)
3. Writing mode is confirmed (suggested by system, overridable by user)

Input: selected theme's full insight extraction + chosen hook + confirmed mode.

---

## Universal Post Structure

Regardless of mode, every post follows this skeleton:

```
[HOOK — from Skill 03]

[BODY — 3–5 paragraphs, mode-specific]

[CLOSE — action, open question, or resonant line — never a summary]
```

Mobile formatting rules (applied to every mode):
- No paragraph longer than 3 lines on a phone screen
- White space is intentional — used between paragraph shifts, not as filler
- First line of each paragraph must carry weight on its own
- No consecutive lines starting with the same word
- End on forward motion or curiosity — never a recap

## Mode 1 — Authority

**Vibe:** The author has already done the thinking. The reader receives the conclusion. No hedging. No "it depends." The author is the guide, the reader is following.

**When the system suggests this:** Theme has a clear directional signal with strong Tier A backing. Non-obvious angle is a firm conclusion, not a question.

### Framework A — Verdict First
```
[Hook]
[The conclusion stated plainly, before any evidence]
[Evidence layer 1 — most specific]
[Evidence layer 2 — broader context]
[What this means going forward]
[Close: the implication most people will miss]
```

### Framework B — The Three Realities
```
[Hook]
[What most people think is true right now]
[What is actually true — contradicts or complicates]
[What will be true in 6–12 months if this continues]
[Close: the single thing worth watching]
```

### Framework C — The Hierarchy of What Matters
```
[Hook]
[The noise: what everyone is focused on]
[The signal: what actually matters]
[The meta-signal: what the signal tells us about the industry's direction]
[Close: direct recommendation or positioning statement]
```

### Framework D — The Non-Negotiables
```
[Hook]
[3 things that are now true regardless of what happens next]
[Why each one matters more than the headline]
[Close: what changes for anyone who accepts these as true]
```

## Mode 2 — Contrarian

**Vibe:** The consensus is stated fairly — then fractured. Not dismissive. Not arrogant. The author sees something the crowd missed and is willing to say so.

**When the system suggests this:** Theme challenges a widely held belief. Non-obvious angle inverts or complicates the mainstream narrative.

### Framework A — Fracture Point
```
[Hook]
[State the consensus view clearly and fairly — 1–2 lines]
[The one data point or development that doesn't fit]
[Why the consensus ignores it or explains it away]
[The alternative frame — what the evidence actually supports]
[Close: the question the consensus can't answer]
```

### Framework B — Right Problem, Wrong Diagnosis
```
[Hook]
[Everyone agrees there is a problem — state it]
[The mainstream explanation for the problem]
[Why that explanation is incomplete or misdirected]
[The actual cause or dynamic]
[Close: what a correct diagnosis changes about the solution]
```

### Framework C — Whose Interests Does This Serve?
```
[Hook]
[The narrative that is being promoted]
[Who benefits if that narrative is believed]
[What the evidence looks like when you remove that incentive]
[Close: what to watch for that would confirm or deny]
```

### Framework D — The Overhyped vs The Underpriced
```
[Hook]
[What the market/industry is overweighting right now]
[What it is correspondingly underweighting]
[Why this imbalance exists — structural, cognitive, or incentive-based]
[Close: where the dislocation creates a real opportunity or risk]
```

## Mode 3 — Analytical

**Vibe:** Structured, precise, evidence-led. Cause → effect chains. Second-order consequences. Ends with a genuine open question that invites professional discussion.

**When the system suggests this:** Theme has a clear mechanism — something happened because of something else, and that cause-effect chain has consequences worth tracing.

### Framework A — The Domino Chain
```
[Hook]
[The first domino — the event that started the sequence]
[Domino 2 — the direct consequence]
[Domino 3 — the consequence of the consequence]
[The last domino — the non-obvious endpoint if the chain completes]
[Close: what would have to break the chain]
```

### Framework B — Signal vs Noise Breakdown
```
[Hook]
[The noise: what most coverage is focused on]
[Why the noise is technically true but strategically irrelevant]
[The actual signal: the development buried in the same story]
[What the signal implies for the next 90 days]
[Close: the metric or event that would confirm the signal]
```

### Framework C — The Before / After
```
[Hook]
[How this part of the industry worked before this development]
[What specifically changed — the decision, announcement, or event]
[How it works now — what's different in practice]
[What hasn't changed yet but likely will]
[Close: the open question that determines how this resolves]
```

### Framework D — The Incentive Map
```
[Hook]
[Who are the key actors in this situation]
[What does each actor actually want — not what they say publicly]
[How do those incentives conflict or align]
[What behaviour those incentives predict going forward]
[Close: the moment when incentives will force a visible decision]
```

## Mode 4 — Strategic

**Vibe:** Written for people who make decisions. Translates a market signal into a forced choice. Symmetrical — what separates the companies that benefit from those that don't. Actionable without being prescriptive.

**When the system suggests this:** Theme has clear business implications. Non-obvious angle is a strategic asymmetry — some actors are positioned better than others.

### Framework A — Winners and Losers
```
[Hook]
[The shift that is creating the asymmetry]
[What the companies that benefit are doing / will do]
[What the companies that get hurt are doing / won't do]
[The specific capability or decision that separates the two groups]
[Close: the window — how long before this asymmetry closes]
```

### Framework B — The Option Nobody Is Pricing
```
[Hook]
[The conventional strategic response to this development]
[Why the conventional response is insufficient or late]
[The non-obvious move that changes the equation]
[Who has the positioning to make that move]
[Close: what it would look like when someone takes it]
```

### Framework C — 30 / 90 / 180 Days
```
[Hook]
[What the smart response looks like in the next 30 days]
[What the 90-day horizon looks like if this develops as expected]
[What the 180-day position looks like for those who acted early vs late]
[Close: the single decision that separates the two trajectories]
```

### Framework D — The Capability Gap
```
[Hook]
[What this development now requires from companies in this space]
[Who already has that capability]
[Who is building it]
[Who doesn't have it and hasn't started]
[Close: what happens to the third group if the timeline accelerates]
```

## Mode 5 — Future Scenario

**Vibe:** Maps futures. Never predicts. Two or three plausible trajectories, each internally consistent. Reader is equipped with a leading indicator to watch.

**When the system suggests this:** Theme involves a branching point — a decision, regulation, or market development that could go multiple ways with meaningfully different outcomes.

### Framework A — Two Worlds
```
[Hook]
[The current trajectory — where things are heading if nothing changes]
[The branch point — the decision or event that determines which path]
[World A: if [condition X] happens — what follows]
[World B: if [condition Y] happens — what follows]
[Close: the specific early signal that tells you which world is forming]
```

### Framework B — The Variable Nobody Controls
```
[Hook]
[The scenario most analysts are modelling]
[The variable that isn't in their model but should be]
[How that variable changes the outcome]
[What happens to the mainstream scenario if it materialises]
[Close: where that variable is visible earliest]
```

### Framework C — Three Timelines
```
[Hook]
[If this accelerates: what the world looks like in 12 months]
[If this stalls: what the world looks like in 12 months]
[If this reverses: what the world looks like in 12 months]
[Close: which timeline currently has the most supporting evidence]
```

## Mode 6 — Narrative

**Vibe:** Story-driven. Opens with a specific human moment. Zooms out to the industry dynamic that created it. Identifies the pattern. Lands with resonance, not analysis. The reader doesn't feel informed — they feel understood.

**When the system suggests this:** Theme has a clear human consequence. Non-obvious angle is about people, decisions, or lived experience rather than market mechanics.

### Framework A — The Scene → Industry
```
[Hook — human scene, specific]
[What industry dynamic put that person/team in that position]
[The pattern: this scene is playing out everywhere, not just there]
[The underlying shift that makes it structural, not anecdotal]
[Close: a line that lands emotionally — not a call to action, a recognition]
```

### Framework B — The Invisible Person
```
[Hook]
[Identify the person most affected by this development who nobody is writing about]
[What their situation looks like in concrete terms]
[What the industry's behaviour looks like from their perspective]
[Close: what it would take for the industry to see them]
```

### Framework C — The Decision That Will Echo
```
[Hook — a specific decision made somewhere this week]
[What the person/team was trying to solve]
[Why this decision is representative of a broader pattern]
[What the ripple effect looks like over time]
[Close: the version of this decision that will be studied in three years]
```

### Framework D — The Gap Between the Story and the Reality
```
[Hook]
[The official story: how the industry talks about what's happening]
[The lived reality: how it actually feels to people inside it]
[What creates the gap — structural, incentive, or communication]
[Close: what would have to change for the story and reality to align]
```

## Mode 7 — Explainer

**Vibe:** The author saves the reader three hours of reading in two minutes. Opens by naming the confusion precisely — not dismissively. Gives a clean, jargon-free definition. Anchors with an example. Closes with urgency: why this matters right now.

**When the system suggests this:** Theme involves a concept, term, mechanism, or development that most of the audience half-understands or misunderstands.

### Framework A — The Backwards Definition
```
[Hook — naming the specific misunderstanding]
[What most people think it means]
[What it actually means — precise, jargon-free]
[A concrete example that makes it stick]
[Close: why understanding it correctly changes how you read this week's news]
```

### Framework B — The 90-Second Version
```
[Hook]
[The whole picture in three lines — the minimum viable understanding]
[The one nuance most 90-second versions miss]
[How that nuance changes the practical implications]
[Close: the one sentence that someone can take away and actually use]
```

### Framework C — The Terminology Gap
```
[Hook — the industry is using a term incorrectly or inconsistently]
[Who uses it which way and why]
[What the correct or precise definition should be]
[Why the confusion matters — it causes real misalignment in decisions]
[Close: the question to ask when you see the term used, to know which meaning applies]
```

## Mode 8 — Tension Report

**Vibe:** Two contradictory signals are presented as intelligence, not confusion. The author doesn't resolve the tension — they make it productive. Reader leaves with a clearer picture of a complex situation and a specific thing to watch.

**When the system suggests this:** TENSION SIGNAL flag was raised in Skill 02. Tier A data and Tier C sentiment point in different directions.

### Framework A — Two Truths
```
[Hook — state the contradiction directly]
[Signal A: what the data/official sources show — specific]
[Signal B: what practitioners/sentiment shows — specific]
[Why both are real — each reflects a different layer of the same situation]
[What that combination historically predicts]
[Close: the event or data point that will tell you which layer wins]
```

### Framework B — Who Sees What
```
[Hook]
[What the situation looks like from the outside — the official picture]
[What the situation looks like from the inside — practitioner view]
[Why the two views diverge — structural, informational, or incentive-based]
[What it means that both are being expressed publicly at the same time]
[Close: which view has historically been the leading indicator]
```

### Framework C — The Question Behind the Contradiction
```
[Hook]
[State both contradictory signals clearly]
[The surface-level question: which one is right]
[The deeper question: why does this contradiction exist at all]
[What answering the deeper question reveals about the industry's structure]
[Close: the thing to watch that addresses the deeper question, not the surface one]
```

## Framework Selection Logic

For each mode, the system selects the framework based on:

| Insight type | Best framework match |
|---|---|
| Clear directional conclusion | Authority A or D |
| Challenges a narrative | Contrarian A or B |
| Reveals incentive structure | Contrarian C or Analytical D |
| Cause-effect chain | Analytical A |
| Buried signal in coverage | Analytical B |
| Business decision implications | Strategic A or C |
| Non-obvious capability gap | Strategic D |
| Multiple future paths | Future Scenario A or C |
| One unknown variable | Future Scenario B |
| Human consequence story | Narrative A or C |
| Industry vs lived reality gap | Narrative D |
| Misunderstood term or concept | Explainer A or B |
| Contradictory signals | Tension Report A or C |

The system does not cycle through frameworks in order. It matches framework to insight every time.

## Rules

- The body never summarises the news. It adds the layer beneath the news.
- Every paragraph must earn its place — if removing it doesn't weaken the post, remove it.
- No fake certainty in future-oriented content. Use "if," "when," "likely," "the evidence suggests."
- No hallucinated facts. Every specific claim traces to a Tier A or Tier B source from Skill 02.
- No generic LinkedIn clichés. Prohibited phrases include: "In today's fast-paced world," "I'm excited to share," "Let that sink in," "This is a game-changer," "The future is now."
- The close is never a summary. It is either a forward-looking implication, a direct question, or a resonant line that makes the reader want to engage.
- The post should not feel like news. It should feel like a well-placed conversation with someone who has been watching this closely for a long time.
